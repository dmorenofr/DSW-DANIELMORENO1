Propiedades Basicas de CCS

ETIQUETA DESCRIPCION SINTAXIS
Colors Con esta propiedad se puede cambiar el color del texto style="color:Red;"
Backgrounds Con esta propiedad se puede definir el fondo ya sea una imagen o un color, para diferentes elementos dentro de la pagina background-color: lightblue - background: url("")
Borders Con esta propiedad se puede especificar el estilo, ancho y el color del borde de un elemento dentro de la pagina border-style: solid;
Margins Con esta propiedad se crea espacio alrededor de algun elemento, con un borde definido, se tiene el control total de las margenes tanto Superior, Inferior, Derecha e Izquierda margin-top: 100px - margin-bottom: 100px - margin-right: 150px - margin-left: 80px
Text Con esta propiedad podemos modificar varios aspectos de texto, como las siguientes: Color, Alineacion, Decoracion, Identacion, Direccion y ETC color: blue - text-align: center - text-decoration: overline - text-indent: 50px - direction: rtl
Fonts Con esta propiedad podemos seleccionar el tipo de fuente, negrita, tamaño y estilo de un texto font-family: "Times New Roman", Times, serif
Height Con esta propiedad podemos modificar el alto de un elemto dentro de la pagina height: 200px
Width Con esta propiedad podemos modificar el ancho de un elemto dentro de la pagina width: 200px
Links Con esta propiedad se puede modificar el color, fuente y fondo de los enlaces link {text-decoration: none;}
Lists Con esta propiedad se selecciona el tipo de lista que se desea list-style-type: circle;
Tables Con esta propiedad se puede modificar las caracteristicas de las tablas, como bordes, colores, celda y ECT table, th, td {border: 1px solid black;}
Position Con esta propiedad se puede especificar el tipo posicionamiento utilizado para un elemento(estático, relativo, fijo, absoluto o adhesivo static {position: static;;}
Overflow Con esta propiedad controlamos el desbordamiento cuando el contenido es muy grande y no se ajusta de manera correca dentro de la pagina div { overflow: visible;}
Float Con esta propiedad controlamos como debe flotar un elemento o en otras palabras el posicionamiento del mismo img {float: none;}
Display Con esta propiedad se puede controlar la visualizacion, para los diferentes elementos HTML, los cuales en su mayoria tienen la propiedad block o inline. display: inline - display: block
